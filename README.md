Welcome to my personal developer portfolio! This repository showcases my professional journey, academic background, internships, and full-stack software developments alongside hardware IoT prototyping.

It features a premium **glassmorphism** design theme with ambient color glows and is fully integrated with a custom, standalone **AI Portfolio Assistant widget** that helps recruiters and visitors explore my profile interactively.

---

## 🤖 AI Portfolio Assistant Widget

The portfolio includes a plug-and-play **AI Portfolio Assistant** embedded directly in the bottom-right corner. It is built using **pure HTML, CSS, and Vanilla JavaScript** (no frameworks, no external heavy libraries).

### Key AI Widget Features:
- **Interactive Action Engine:** The AI doesn't just chat; it performs actions in the portfolio! Clicking its custom buttons will scroll visitors to sections (e.g. `#projects`, `#contact`), launch WhatsApp/Email, or trigger my resume modal.
- **Session Name Memory:** Dynamically captures and remembers the visitor's name during their browser session, responding in a friendly, conversational manner.
- **Direct Match Overrides:** Includes specialized dialog pipelines for recruiter inquiries (strengths, fresher status, why hire), fun queries (jokes, coffee/tea, dark mode preferences), and social link redirects.
- **Sleek Interface:** Neon-themed glassmorphism panel, typing animations, word-by-word streaming responses, and scrollbar layouts.

## 🚀 How to Run Locally

Since the portfolio and the AI Assistant are built using static frontend assets, you can run them directly:

1. Clone this repository:
   ```bash
   git clone https://github.com/Praveenkumarmelapu/portfolio.git
   ```
2. Navigate to the project folder and open `index.html` in your browser.
3. *Alternative:* Use VS Code extension **Live Server** to run it locally on `http://127.0.0.1:5500`.
