AI Portfolio Assistant

This is a standalone widget demo. The floating AI assistant has been injected at the bottom-right corner of the page.

To integrate this widget into your portfolio, add these lines:

<!-- 1. Add style sheet in <head> -->
<link rel="stylesheet" href="ai-assistant.css">

<!-- 2. Add widget target container in <body> -->
<div id="ai-assistant"></div>

<!-- 3. Add script file before </body> -->
<script src="ai-assistant.js"></script>
